Проект Города-Храма


Плане превращения города, либо района в Храм.

1. Выделить регион, где живут великие люди, историчесие деятели, либо где такие проживали. Учесть какие там есть объекты культуры, исторические сооружения, памятники, монументы. Форма этого района должна быть правильноя. В идеале - круг, но можно и восьмиугольник, или что-то подобное.
2. Обнести этот район стенами в 1.5-2 этажа в высоту, соединив с домами, в дорогах, с учетом тротуаров, предусмотреть арки. Стена должна быть из белого, либо желтого камня (кирпича). Предусмотреть возможность подъема стену для обзора, прогулок, живописи, фото, закладки подарков, цветор. Подъем по лесенкам. Должны быть перила, чтобы никто не упал.

3. Возле реки, если она есть - стена должна быть по пояс.
